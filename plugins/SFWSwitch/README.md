# SFW Switch

https://discourse.stashapp.cc/t/sfw-switch/4657

## Features

- Adds a SFW toggle button to the menu bar.
  - Green = Blur enabled
  - Gray = Blur disabled
- Toggling the button blurs cover images and other content.
- Hovering over an image temporarily removes the blur.
